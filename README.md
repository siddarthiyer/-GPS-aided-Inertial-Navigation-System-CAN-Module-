# GPS-aided-Inertial-Navigation-System-Module

* MCU: STM32F103C8T6
* Communication Protocol: CAN

Sensors:

* MPU-9250
  * 9-axis MEMS sensor
  * Pinout:
    - PB11: SDA
    - PB10: SCL
  * Communication Protocol: I2C
  
* NEO-6M
  * GPS Module
  * Pinout:
    - PA3: TX
    - PA2: RX
  * Communication Protocol: UART
 
* BMP180
  * Barometric Pressure/Temperature/Altitude Sensor
  * Pinout:
    - PB7: SDA
    - PB6: SCL
  * Communication Protocol: I2C
